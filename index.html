SW Destiny Proxy Maker

About
I made this proxy generator for people who want to try out cards from SW Destiny before buying them.
This open-source project was made by @ossianboren (github link) using an API from swdestinydb.com.

Instructions
Add cards by their ID number. Then print from your browser (they should be neatly gridded for cutting).

Custom Database
If you download this page, you can provide your own database! Just get the cards.json from here, put it in the SWDestinyProxy folder, then change cards.json however you like! (Translations, custom cards, etc)
